MIT-MAN – Freiheit frisst Bürokratie

START
1. Entpacken Sie die ZIP-Datei vollständig.
2. Doppelklicken Sie auf „MIT-MAN starten.cmd“ oder direkt auf „MIT-MAN.html“.
3. Das Spiel läuft lokal im Standardbrowser. Eine Internetverbindung ist nicht erforderlich.

STEUERUNG
- Pfeiltasten oder W/A/S/D: bewegen
- Leertaste: Pause / weiter
- M: Ton an / aus
- Enter: starten bzw. nach Spielende neu starten
- Alternativ stehen Schaltflächen für Maus und Touchscreen bereit.

SPIELZIEL
Alle Punkte im Labyrinth einsammeln. Die vier großen Kraftpunkte machen die Gegner vorübergehend verwundbar.

TECHNIK
Das Spiel besteht aus einer einzigen eigenständigen HTML-Datei. Spielstand und Highscore bleiben ausschließlich lokal im Browser.
